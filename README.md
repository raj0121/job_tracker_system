# 🚀 Job Tracking System (ATS)

A full-stack **Applicant Tracking System (ATS)** designed to streamline recruitment workflows, manage candidates, and track hiring stages efficiently.

---

## 📌 Overview

This project helps organizations manage the complete hiring process — from job posting to candidate selection — in a structured and scalable way.

It includes real-time tracking, structured pipelines, and a clean dashboard interface for better decision-making.

---

## 🛠️ Tech Stack

### Frontend

* React.js (TypeScript)
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js
* REST APIs

### Database

* SQLite / MongoDB (based on your setup)

### Authentication

* JWT (JSON Web Token)

---

## ✨ Features

* 📄 Job Posting Management
* 👤 Candidate Tracking System
* 🔄 Stage-wise Recruitment Pipeline
* 📊 Real-time Dashboard
* 🔐 Secure Authentication (JWT)
* 📁 Organized Data Flow (Frontend ↔ Backend)
* ⚡ Scalable Architecture

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/raj0121/job-tracking-system.git
cd job-tracking-system
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
npm run dev
```

---

### 3️⃣ Frontend Setup

```
cd frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside the backend folder:

```
PORT=5000
JWT_SECRET=your_secret_key
DB_URL=your_database_url
```
